# 症状メモ（Symptom Notes）

シンプルに「毎日テキストで症状を記録」する iOS アプリです。見た目と使い心地にこだわり、ハプティクスや控えめなサウンド演出を入れています。

このリポジトリには Xcode でそのまま使える SwiftUI コード一式（SwiftDataベース）が入っています。Windows 上ではビルドできないため、後述の手順で Mac の Xcode に取り込んでください。

## 主要機能
- 毎日のメモ（1日1件を想定。既存があれば編集）
- 美しいグラデーション背景とカードUI
- ハプティクス（成功/失敗/軽いタップ）
- 控えめなシステムサウンド（トック音）
- 設定（ハプティクス/サウンド ON/OFF、アクセントカラー）
- すべてローカル保存（SwiftData）

## 対応環境
- iOS 17 以降（SwiftData を使用）
- Xcode 15 以降を推奨

## 取り込み手順（Mac + Xcode）
1. このフォルダを Git でクローンまたはZipで取得し、Mac にコピーします。
2. Xcode を開き、File > New > Project... から "App"（iOS）を選択。
3. Interface: SwiftUI / Language: Swift / Storage: 何でも可（後からSwiftDataを有効化）
4. プロジェクト作成後、プロジェクトのターゲットを iOS 17+ に設定。
5. このリポジトリの `ios/` フォルダ内のファイルを、Xcode ナビゲータにドラッグ&ドロップで追加（"Copy items if needed" にチェック）。
6. ビルド＆実行（⌘+R）。

補足: 初回は SwiftData 用に iOS17ターゲット/最小SDKの調整、Signing & Capabilities のチーム選択が必要です。

## アプリの使い方
- 右上の + ボタンで「今日のメモ」を作成/編集
- リストから日付をタップして過去メモを編集
- 左上の歯車から設定を開き、ハプティクス・サウンド・アクセントカラーを切替

## デザインメモ
- メインカラーはベージュ基調（落ち着き・温かみ）
- カードは .ultraThinMaterial + 角丸 + やわらかい影で上質感
- 背景はベージュの柔らかいグラデーション（軽量）
- 文字はSF Proのダイナミックタイプに準拠、可読性優先

## 公開（App Store）までの流れ 概要
1. Apple Developer Program 登録（有料）
2. アプリアイコン・スクショ・プライバシーポリシー準備
3. バージョン/ビルド番号設定、アプリアーカイブ（Product > Archive）
4. Xcode Organizer から TestFlight へアップロード
5. App Store Connect でメタデータ入力、審査提出

## 次の拡張候補
- メモの検索・タグ
- カレンダービュー（月カレンダーから日付選択）
- ウィジェット/ロック画面コンプリケーション
- iCloud 同期（CloudKit + SwiftData）

## ブランド化（アイコン/配色）提案メモ
- アイコンモチーフ: ハート×メモ帳、やわらかいトーン（温かみ）
- ベースカラー: ベージュ系（#C19A6B を基調、補助に #EED9C4 / #A67B5B）
- アクセントに穏やかなグリーンやコーラルを少量使用
- ライト/ダーク両対応でコントラストと可読性を優先
